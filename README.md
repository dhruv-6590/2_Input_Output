# 2_Input_Output
The first code [2_Input_Output.py] includes the basic knowledge of how an input() function takes input from the user and print back on the screen with Hello (Your Name) Welcome!.
The Second code [3_Printing_Variables.py] uses the print() function to print single and multiple variables. We can print multiple variables by separating them with commas.
The Third code [4_Mul_Input_plus_split.py] allows us to take multiple values in a single line and separate their respective variable values using split().
Since input() returns a string by default, in Fourth code [5_Typecasting_in_input.py] we can convert it into other data types like int and float by typecasting.
